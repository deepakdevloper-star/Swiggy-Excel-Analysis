# 🍽️ Swiggy Food Order Data Cleaning & Analysis

## Project Overview

This project demonstrates the cleaning and analysis of Swiggy food order data using Microsoft Excel. The raw order data was transformed into a structured dataset, followed by summary analysis using Pivot Tables.

## Skills Demonstrated

* Data Cleaning
* Text Splitting
* Date Formatting
* Pivot Tables
* Data Analysis

## Project Workflow

1. Cleaned raw order data.
2. Extracted Order ID, Date, Item Name, Quantity, Price, Customer Type, Feedback, Discount, and Delivery Location.
3. Standardized formatting.
4. Created Pivot Tables to identify top-performing items and order trends.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Excel Functions

## Outcome

The project provides meaningful insights into customer orders, popular food items, and sales performance.
